O Magic tem como objetivo ajudar empresas de todos os tamanhos a construir e implantar rapidamente aplicações de IA para alcançar um aumento de produtividade de 100x.

## Matriz de Produtos Magic

O Magic é a primeira **"plataforma de produtividade com IA all-in-one open source"**, não um único produto de IA, mas uma matriz de produtos abrangente com capacidades ricas.

![Matriz de Produtos](https://public-cdn.letsmagic.ai/static/img/super-magic-open-source-projects-en.png?v=20250819)

- **[Super Magic](https://github.com/tassomuniz/super-magic)** - Um **Agente de IA de propósito geral** projetado para cenários de tarefas complexas
- **[Magic IM](https://github.com/tassomuniz/magic)** - Um sistema de mensagens instantâneas de nível empresarial que integra conversações de Agente de IA com comunicação interna empresarial
- **[Magic Flow](https://github.com/tassomuniz/magic)** - Um poderoso sistema de orquestração visual de fluxos de trabalho de IA
- **Teamshare OS** (Em breve) - Um sistema de escritório colaborativo online de nível empresarial

Além dos produtos de IA acima, também open-sourced algumas das infraestruturas que usamos para construir esses produtos:

- **[Agentlang](https://github.com/tassomuniz/agentlang)** - Um Framework de Agente de IA baseado em linguagem para construir agentes de IA com linguagem natural (atualmente disponível na versão Python, versão TypeScript em breve)
- **[Magic Lens](https://github.com/tassomuniz/magiclens)** - Uma ferramenta poderosa e flexível de conversão HTML para Markdown que usa um sistema de regras extensível para converter com precisão documentos HTML complexos para formato Markdown conciso
- **Magic Use** (Em breve) - Uma ferramenta revolucionária de operação de navegador projetada especificamente para Agentes de IA
- **Magic Space** (Em breve) - Um novo sistema de gerenciamento de hospedagem de conteúdo estático projetado especificamente para Agentes de IA
- **Sandbox OS** (Em breve) - Um poderoso sistema de sandbox para runtime de Agente de IA

### Super Magic

Um poderoso **Agente de IA de propósito geral** especialmente projetado para cenários de tarefas complexas. Através de um sistema de design multi-agente e capacidades de ferramentas ricas, o Super Magic suporta habilidades inteligentes como **compreensão autônoma de tarefas**, **planejamento autônomo de tarefas**, **ação autônoma** e **correção autônoma de erros**. Ele pode entender instruções em linguagem natural, executar vários processos de negócios e entregar resultados finais desejados. Como produto principal da matriz de produtos Magic, o Super Magic fornece poderosas capacidades de desenvolvimento secundário através de código aberto, permitindo que empresas construam e implantem rapidamente assistentes inteligentes que atendam às necessidades específicas do negócio, melhorando muito a eficiência e qualidade da tomada de decisões.

### Magic Flow

Magic Flow é um poderoso sistema de orquestração visual de fluxos de trabalho de IA que permite aos usuários construir fluxos de trabalho complexos de Agente de IA em uma tela livre. Ele tem as seguintes características principais:

- **Orquestração Visual**: Interface intuitiva de arrastar e soltar permite projetar fluxos de trabalho de IA complexos sem codificação, implementando facilmente várias combinações funcionais através de conexões de nós.
- **Biblioteca de Componentes Rica**: Variedade integrada de componentes predefinidos, incluindo processamento de texto, geração de imagens, módulos de execução de código, atendendo diversas necessidades de negócios.
- **Suporte Abrangente a Modelos**: Compatível com qualquer modelo grande seguindo o protocolo API OpenAI, escolhendo flexibilmente capacidades de IA adequadas para cenários de negócios.
- **Capacidade de Integração de Sistema**: Integração perfeita com Magic IM e outros sistemas de IM de terceiros (WeCom, DingTalk, Feishu), habilitando colaboração multiplataforma.
- **Extensões Personalizadas**: Suporte para desenvolvimento de nós de ferramentas personalizadas para atender requisitos específicos de cenários de negócios.
- **Depuração e Monitoramento em Tempo Real**: Fornecendo funções abrangentes de depuração e monitoramento para ajudar a identificar e resolver rapidamente problemas em fluxos de trabalho, garantindo operação estável de aplicações de IA.

### Magic IM

Magic IM é um sistema de conversação de Agente de IA de nível empresarial projetado especificamente para cenários de gerenciamento de conhecimento interno e atendimento inteligente ao cliente. Ele fornece capacidades conversacionais ricas, suportando diálogos multi-turnos, compreensão de contexto, recuperação de base de conhecimento e outras funções, permitindo que empresas construam rapidamente atendimento inteligente ao cliente, assistentes de conhecimento e outras aplicações.

Magic IM tem as seguintes características principais:

- **Gerenciamento de Base de Conhecimento**: Funções poderosas de gerenciamento de base de conhecimento, suportando importação de vários formatos de documento, indexação automática e recuperação semântica, garantindo respostas de IA baseadas em conhecimento empresarial autêntico.
- **Gerenciamento de Conversas**: Gerenciamento abrangente de conversas, suportando distinção de tópicos para diferentes conteúdos de conversa, habilitando tanto conversas de Agente de IA quanto comunicação com pessoas dentro da organização.
- **Capacidade de Chat em Grupo**: Funcionalidade poderosa de chat em grupo, suportando discussões colaborativas em tempo real entre várias pessoas, com IA participando inteligentemente em chats de grupo e fornecendo respostas instantâneas, promovendo comunicação eficiente da equipe e compartilhamento de conhecimento.
- **Arquitetura Multi-organizacional**: Suporte para implantação multi-organizacional e isolamento estrito de dados organizacionais, com cada organização tendo espaço de dados independente e permissões de acesso.
- **Segurança de Dados**: Mecanismos rigorosos de isolamento de dados e controle de acesso, gerenciamento de permissões multi-nível, protegendo informações empresariais sensíveis e garantindo que não haja vazamento de dados entre organizações.

## Teamshare OS

Teamshare OS é uma plataforma de escritório colaborativo de nível empresarial moderna projetada para melhorar a eficiência da colaboração em equipe e gerenciamento de conhecimento. Como componente importante da matriz de produtos Magic, Teamshare integra profundamente capacidades de IA em cenários de escritório diários, alcançando fluxos de trabalho inteligentes e gerenciamento de conhecimento.

Teamshare OS tem as seguintes características principais:

- **Gerenciamento Inteligente de Documentos**: Suporte para edição online, colaboração e controle de versão de vários formatos de documento, geração e otimização de conteúdo assistida por IA, tornando o gerenciamento de documentos da equipe mais eficiente.
- **Magic Table**: Ferramenta poderosa de gerenciamento de dados multidimensional, suportando tipos de campo personalizados, visualizações diversas e fluxos de trabalho automatizados, combinados com capacidades de IA para alcançar processamento inteligente de dados, atendendo diversas necessidades de gerenciamento de dados.
- **Gerenciamento de Colaboração de Projetos**: Quadros de projeto intuitivos e gerenciamento de tarefas, suportando fluxos de trabalho personalizados, combinados com análise inteligente de IA para fornecer previsão de progresso de projeto e sugestões de otimização de recursos.
- **Base de Conhecimento**: Sistema poderoso de consolidação e recuperação de conhecimento, estruturando automaticamente documentos internos empresariais para formar ativos de conhecimento empresarial acumulados sustentáveis.
- **Capacidade de Integração Abrangente**: Integração perfeita com matriz de produtos Magic, enquanto suporta conexão com software de escritório mainstream e aplicações empresariais, criando uma plataforma de trabalho unificada.

### Edição Comunitária Auto-hospedada

#### Requisitos do Sistema
- Docker 24.0+
- Docker Compose 2.0+

#### Iniciar o Sistema Usando Docker

```bash
# Clonar repositório
git clone https://github.com/tassomuniz/magic.git
cd magic

# Iniciar serviço em primeiro plano
./bin/magic.sh start
```

##### Outros Comandos

```bash
# Iniciar serviço em segundo plano
./bin/magic.sh daemon

# Verificar status do serviço
./bin/magic.sh status

# Visualizar logs
./bin/magic.sh logs
```

##### Configurar Variáveis de Ambiente

```bash
# Configurar variáveis de ambiente do Magic, deve configurar pelo menos variáveis de ambiente de um modelo de linguagem grande para usar o Magic normalmente
cp .env.example .env

# Configurar variáveis de ambiente do Super Magic, deve configurar qualquer modelo de linguagem grande que suporte formato OpenAI para usar normalmente
./bin/magic.sh status
cp config/.env_super_magic.example .env_super_magic
```
